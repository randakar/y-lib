\subsection{Detailed Description}
Standard C function headers. This header file contains the most used 'standard C functions' in Vugen scripts. \par
By declaring these functions properly the compiler can check if these functions are used correctly and apply implicit conversions when necessary. \par
This header file 'de-\/strips' the compiler in Vugen. \par
It contains all functions mentioned in the H\-P Load\-Runner Online Function Reference. \begin{DoxyNote}{Note}
Sites like \href{http://www.cplusplus.com/reference/clibrary/}{\tt http\-://www.\-cplusplus.\-com/reference/clibrary/} explain the standard C functions better than the H\-P Load\-Runner Online Function Reference and more.
\end{DoxyNote}

\begin{DoxyCode}
\textcolor{preprocessor}{#include "\hyperlink{vugen_8h}{vugen.h}"} 
\end{DoxyCode}
 Or\-: add to include-\/directory and add the \#include to lhrun.\-h on all Vugen machines and load generators. \par
