\subsection{Detailed Description}
V\-T\-S functions for L\-R 11.\-5 and above. Library wrapperrs for talking to a V\-T\-S-\/server. This is part of L\-R 11.\-5 and needs to be installed seperately. For L\-R versions older than 11.\-5, see the old \hyperlink{y__vts_8c}{y\-\_\-vts.\-c}.

\begin{DoxyWarning}{Warning}
This \hyperlink{y__vts_8c}{y\-\_\-vts.\-c} is N\-O\-T compatible with the vts version that was shipped before L\-R 11.\-5, and the older \hyperlink{y__vts_8c}{y\-\_\-vts.\-c} is N\-O\-T compatible with the version in L\-R 11.\-5 either. 
\end{DoxyWarning}
